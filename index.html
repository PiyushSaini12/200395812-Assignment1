Let's play the game of See N' Say storyteller!




Cheat Sheet


  • To use the See N' Say storyteller, kindly tap on the buttons on the right side of the book.
  • Different Buttons triggers a Different set of series on the left side of the book.
  • As, Use Purple Button for Nouns, Blue for Verbs, Green for Adjectives, Orange for Nouns and Pink for Places.
  • Use the 'Surprises' button (Red) for a random surprise.
  • Use the 'Playback' button (Ocean-green) for the last repeated sentence.
  • Use the 'Reset' button (Brown) to empty the playback and start from the New!
